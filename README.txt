Projekt: Excel-Lernpfad
Erstellt von: Lisi Kemminger und Nimaro ChatGPT 4o

Dieses Projekt enthält:
- Eine React-App mit semantischem Suchfeld
- Unterbuttons zu jedem Register der Datei 'Allgemeine_Tipps.xlsx'
- Einfach formulierte Erklärungen für Excel-Anfänger
- Integrierte Screenshots

Anleitung zum Hochladen:
1. Repository auf GitHub erstellen
2. Projektordner auf GitHub hochladen
3. In Vercel mit GitHub verknüpfen und deployen
